== README

Proyecto PYS correspondiente al ramo Proyecto de Base de Datos de la carrera 
Ingeniería Civil Informática de la Universidad de Santiago de Chile.

Desarrollado por tres estudiantes de dicha carrera: Alexis Casanueva, Claudia
Guzmán y Camila Marín.

Dentro del repositorio se puede encontrar la aplicación, los triggers y un 
backup de la base de datos con el cual se puede realizar el poblado de esta,
para realizar las pruebas correspondientes.
